## Summary
"Oscar P's Journey" is a simple platformer that follows Tarleton State University's mascot. Traverse through 4 levels with varying themes, obstacles, and easter eggs for Tarleton students and alumni. 

## Context
Oscar P's Journey is a platformer developed by Tarleton Computer Society  Students during the spring 2025 semester. Students were led by Spring 2025 Game Development Officer Dillon Gatlin to create levels for Tarleton's beloved mascot, "Oscar P,"  to traverse.  Each level was designed by one student, showcasing their talents, creativity, and programming skills. Students were given free rein on themes and features as long as Oscar P was the player character.
